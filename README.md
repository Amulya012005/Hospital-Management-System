# Hospital Management System

# Overview
A web-based Hospital Management System developed using Flask and SQLite to efficiently manage patients, doctors, and appointment records through an organized and user-friendly interface.

# Features
- Register and manage patient records
- Add and manage doctor information
- Schedule and view appointments
- Dashboard displaying patient and doctor statistics
- SQLite database integration for data management
- Clean and responsive user interface

# Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS
